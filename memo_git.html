• git init
  • git status
  • git add + name of the file, ex: git add index.html
  • git commit -m "Message that explains the changes"
  • git log
  • git add . :adds all changes
  • git checkout , allows you to come back to an older version
  • git branch : list the branches
  • git branch master : reloads the master branch
  • git branch + name of the new branch: creates a new branch
  • git branch -m 'newname' : rename the branch
  • git branch -d (ou -D) nameofthebranch : delete the branch
  • git merge + name of the branch to merge into master
  • git merge -s ours master : merge master into another branch
